## 演示流程（5 分钟）

### 第一幕：开场（30 秒）

**画面**：首页 hero 区

**话术**：
> "大家好，这是『邻里圈』—— 一个 AI 驱动的社区互助平台。我们希望用技术让邻里关系重新温暖起来。"

**动作**：
- 滚动展示首页
- 突出 hero 区的 AI 标签

---

### 第二幕：四大模块快速浏览（60 秒）

**画面**：依次点击四大模块入口

**话术**：
> "邻里圈提供四大核心服务：技能交换、共享厨房、时间银行、应急邻里。每一个都对应一个真实的社区场景。"

**动作**：
1. 点击"技能交换" → 展示列表
2. 返回 → 点击"共享厨房" → 展示
3. 返回 → 点击"时间银行" → 展示
4. 返回 → 点击"应急邻里" → 展示

---

### 第三幕：AI 智能推荐（90 秒，重点）

**画面**：技能交换详情页

**话术**：
> "这是我们的核心亮点——AI 三维智能匹配。传统平台只能关键词匹配，我们结合了**语义理解**、**地理位置**、**信誉权重**三个维度。"

**动作**：
1. 进入一个技能详情（如"钢琴教学"）
2. 滚动到底部 → 展示 **AI 推荐区块**
3. **逐个解读推荐卡片**：
   - "匹配度 87% —— AI 通过语义识别知道他会弹琴"
   - "信誉 4.8 星 —— 系统给高信誉用户更高权重"
   - "距离 1.2 km —— 10 公里内线性衰减"
4. 点击"**降级演示**"：临时禁用 AI_KEY，刷新页面展示降级提示

**技术展示**（可选）：
> "背后调用大模型 API 实现语义匹配，未配置时自动降级为类别匹配，保证 100% 可用。"

---

### 第四幕：应急智能推送（90 秒）

**画面**：应急求助发布流程

**话术**：
> "应急场景是社区互助的硬需求。我们实现了**300 米智能推送**——求助发布后，系统自动通知地理围栏内的所有响应者。"

**动作**：
1. 进入应急模块 → 点击"+"
2. 填写："突发情况，楼下漏水，急需维修"
3. 选择位置
4. 发布
5. 切换到另一个测试账号 → 展示收到推送
6. 点击推送 → 进入详情 → 响应

**对比说明**（话术）：
> "相比微信群、打电话的传统方式，我们的响应速度从 5-30 分钟降到 1 分钟内。"

---

### 第五幕：时间银行家庭池（60 秒）

**画面**：时间银行家庭绑定页

**话术**：

## 录制注意事项

1. **节奏**：保持每幕 60-90 秒，不要超长
2. **声音**：清晰的旁白 + 配乐（轻快）
3. **画面**：录制前确认鼠标点击流畅、不卡顿
4. **字幕**：关键术语配字幕（AI 智能匹配、300 米推送、家庭时间池）
5. **配色**：突出品牌绿（`#10b981`）与各模块主题色

## 常见 Q&A（路演准备）

**Q: AI 推荐会不会推荐不合适的？**
A: 有信誉分权重 + 用户举报机制，恶意用户会被降权。

**Q: 应急场景如何避免误报？**
A: 有虚假举报审核机制，核实后扣减信誉分。

**Q: 时间银行会通胀吗？**
A: 有日收益上限（480 分钟）+ 时间转账功能。

**Q: 商业模式？**
A: 增值服务（高级功能）+ 数据洞察（B 端）+ 社区团购分成。

**Q: 与闲鱼/UU 跑腿的区别？**
A: 我们是**多场景融合**+**AI 智能**+**代际互助**，其他是单点切入。
