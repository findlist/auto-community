# 「邻里圈」设计规范（Design System）

> 本规范定义了项目统一的设计语言（Design Token），所有前端开发必须遵循。

---

## 一、设计理念

- **温度优先**：色彩选择温暖的中性绿（Primary），传递"邻里互助"的情感价值
- **层次清晰**：通过间距、字号、阴影构建清晰的信息层级
- **响应一致**：所有交互遵循统一的动效曲线与时长
- **包容可达**：满足 WCAG 2.1 AA 标准，支持减弱动画偏好

### 2.2 字体（Typography）

| Token | 值 | 用途 |
|-------|-----|------|
| `--font-size-xs` | `0.75rem` | 辅助说明、版权 |
| `--font-size-sm` | `0.875rem` | 表格内文、副标题 |
| `--font-size-base` | `1rem` | 正文 |
| `--font-size-lg` | `1.125rem` | 卡片标题、强调正文 |
| `--font-size-xl` | `1.25rem` | 区块标题 |
| `--font-size-2xl` | `1.5rem` | 弹窗标题 |
| `--font-size-3xl` | `1.875rem` | 页面主标题 |

字重：`--font-weight-normal/medium/semibold/bold`（400/500/600/700）

### 2.3 间距（Spacing）

基于 4px 基准，使用 `--space-1` 到 `--space-20`。
- 紧凑场景：`--space-1` ~ `--space-2`（4-8px）
- 卡片内边距：`--space-4` ~ `--space-6`（16-24px）
- 区块间距：`--space-8` ~ `--space-12`（32-48px）

### 2.4 圆角（Radius）

| Token | 值 | 适用场景 |
|-------|-----|----------|
| `--radius-sm` | `4px` | 输入框、小标签 |
| `--radius-md` | `8px` | 按钮、小卡片 |
| `--radius-lg` | `12px` | 卡片、模态框 |
| `--radius-xl` | `16px` | 大卡片、Hero 区 |
| `--radius-2xl` | `24px` | 头像、强调元素 |
| `--radius-full` | `9999px` | 胶囊按钮、头像 |

### 2.5 阴影（Shadow）

| Token | 适用场景 |
|-------|----------|
| `--shadow-xs` | 表格行 hover |
| `--shadow-sm` | 卡片默认态 |
| `--shadow-md` | 卡片 hover、下拉菜单 |
| `--shadow-lg` | 弹窗、模态框 |
| `--shadow-xl` | 浮动操作按钮 |
| `--shadow-glow` | 品牌发光效果 |

### 2.6 动效（Transition）

| Token | 值 | 用途 |
|-------|-----|------|
| `--transition-fast` | `150ms ease-out` | 颜色变化、按钮反馈 |
| `--transition-base` | `250ms ease-out` | 卡片 hover、面板展开 |
| `--transition-slow` | `400ms ease-out` | 模态框、页面切换 |

## 三、组件使用规范

### 3.1 按钮

- 主按钮：背景 `--color-primary-500`，hover `--color-primary-600`，文字白色，圆角 `--radius-md`
- 次按钮：背景 `--color-neutral-0`，边框 `--color-neutral-200`，文字 `--color-neutral-700`
- 危险按钮：背景 `--color-error`，文字白色
- 禁用态：背景 `--color-neutral-200`，文字 `--color-neutral-400`

### 3.2 卡片

- 默认：背景白、圆角 `--radius-lg`、阴影 `--shadow-sm`
- Hover：阴影升级 `--shadow-md`，过渡 `--transition-base`

### 3.3 文字层级

- H1：30px / semibold / 主标题
- H2：24px / semibold / 区块标题
- H3：20px / semibold / 卡片标题
- Body：16px / normal / 正文
- Caption：14px / normal / 辅助说明
- Tiny：12px / normal / 时间戳、版权

### 3.4 表单

- 输入框高度：44px
- 圆角：`--radius-md`
- 边框：1px `--color-neutral-200`，focus 时 `--color-primary-500`
- 错误态：边框 `--color-error`，下方红色提示文字

## 四、可访问性（A11y）要求

1. **对比度**：正文与背景对比度 ≥ 4.5:1（WCAG AA）
2. **焦点可见**：键盘 Tab 焦点需有清晰视觉反馈（已通过全局 `:focus-visible` 实现）
3. **减弱动画**：尊重 `prefers-reduced-motion` 媒体查询（已在 index.css 实现）
4. **图片替代**：所有 `<img>` 必填 `alt` 属性
5. **按钮语义**：图标按钮必填 `aria-label`
6. **键盘导航**：Tab 顺序需符合视觉顺序

## 五、命名约定

- CSS 变量：`--{类别}-{名称}`，如 `--color-primary-500`、`--space-4`
- 组件类名：PascalCase 文件名 + BEM 风格（可选），如 `Card`、`EmptyState`
- 工具类：Tailwind 原子类为主，自定义类前缀化（如 `.stagger-item`）

## 六、使用方式

### 6.1 在 CSS 中使用

```css
.button-primary {
  background: var(--color-primary-500);
  color: var(--color-text-inverse);
  border-radius: var(--radius-md);
  box-shadow: var(--shadow-sm);
  transition: all var(--transition-base);
}
```

### 6.2 在 Tailwind 中使用

Tailwind 4 支持直接读取 CSS 变量，也可通过 `theme()` 引用：

```html
<button class="bg-[var(--color-primary-500)] hover:bg-[var(--color-primary-600)] text-white rounded-md shadow-sm">
  主按钮
</button>
```

或使用 Tailwind 默认色板（推荐，避免硬编码）：

```html
<button class="bg-emerald-500 hover:bg-emerald-600 text-white rounded-md">
  主按钮
</button>
```
