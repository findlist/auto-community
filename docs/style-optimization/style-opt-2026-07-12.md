## 三、具体修改内容

### 1. Auth 页面背景图一致性修复

**问题**：Login 页有温暖邻里氛围背景图，但 Register / ForgotPassword / ResetPassword 仅有暗化遮罩、无背景图，导致认证流程视觉割裂。

**修改**：为 Register、ForgotPassword（表单态 + 成功态）、ResetPassword（表单态 + 成功态）补齐与 Login 完全一致的背景图，并补充语义化注释（区分「背景图」与「暗化遮罩」）。

**涉及文件**：
- [Auth/Register.tsx](file:///e:/work/auto-community/client/src/pages/Auth/Register.tsx)
- [Auth/ForgotPassword.tsx](file:///e:/work/auto-community/client/src/pages/Auth/ForgotPassword.tsx)
- [Auth/ResetPassword.tsx](file:///e:/work/auto-community/client/src/pages/Auth/ResetPassword.tsx)

### 2. 模块页面应用设计令牌中的模块色

**问题**：`index.css` 已定义四大模块主题色（`--color-module-skill` 蓝、`--color-module-kitchen` 橙、`--color-module-timebank` 紫、`--color-module-emergency` 红），但各模块列表页的眉题（「—— 模块名」）统一使用中性灰 `text-neutral-400`，模块身份辨识度不足。

**修改**：将各模块页面眉题颜色从中性灰改为对应模块色，强化视觉身份：

| 模块 | 设计令牌 | 色值 |
| --- | --- | --- |
| 技能交换 | `--color-module-skill` | #3b82f6 蓝 |
| 共享厨房 | `--color-module-kitchen` | #f97316 橙 |
| 时间银行 | `--color-module-timebank` | #8b5cf6 紫 |

**涉及文件**：
- [SkillExchange/index.tsx](file:///e:/work/auto-community/client/src/pages/SkillExchange/index.tsx#L127)
- [SharedKitchen/index.tsx](file:///e:/work/auto-community/client/src/pages/SharedKitchen/index.tsx#L187)
- [TimeBank/index.tsx](file:///e:/work/auto-community/client/src/pages/TimeBank/index.tsx#L42)

### 3. TimeBank 快捷入口配色对齐模块色

**问题**：TimeBank 页面的二级功能入口卡片（时间账户、我的订单、亲情绑定）使用翡翠绿（emerald）作为图标背景与悬停色，与时间银行模块紫主题不一致。

**修改**：将快捷入口卡片的图标背景、图标颜色、悬停边框由 emerald 系改为 violet 系（`bg-violet-50` / `text-violet-600` / `hover:border-violet-200`），与模块紫主题统一。

**涉及文件**：[TimeBank/index.tsx](file:///e:/work/auto-community/client/src/pages/TimeBank/index.tsx#L77-L92)

### 4. Admin 移动端抽屉动画与激活态指示

**问题**：
1. 移动端侧边栏抽屉打开时无过渡动画，遮罩与抽屉瞬间出现，体验生硬。
2. 导航激活态仅有背景色变化，位置指示不够明确。

**修改**：
1. 在 `index.css` 新增 `slideInLeft` 关键帧与 `.animate-slide-in-left` 工具类，为抽屉添加左侧滑入动画；遮罩添加 `animate-fade-in` 淡入。
2. 导航激活项增加左侧翡翠绿色条（`absolute left-0 top-1.5 bottom-1.5 w-0.5`），作为编辑式位置标记，强化当前位置指示。

**涉及文件**：
- [index.css](file:///e:/work/auto-community/client/src/index.css#L227-L254)（新增 slideInLeft 关键帧与工具类）
- [Admin/AdminLayout.tsx](file:///e:/work/auto-community/client/src/pages/Admin/AdminLayout.tsx#L29-L55)（激活态色条）
- [Admin/AdminLayout.tsx](file:///e:/work/auto-community/client/src/pages/Admin/AdminLayout.tsx#L88-L99)（抽屉动画）

## 四、优化前后对比说明

### 4.1 Auth 流程一致性

| 维度 | 优化前 | 优化后 |
| --- | --- | --- |
| Login 背景 | 温暖邻里窗台图 ✓ | 不变 |
| Register 背景 | 仅暗化遮罩，无图 ✗ | 补齐同款背景图 ✓ |
| ForgotPassword 背景 | 仅暗化遮罩（表单态+成功态） ✗ | 两态均补齐 ✓ |
| ResetPassword 背景 | 仅暗化遮罩（表单态+成功态） ✗ | 两态均补齐 ✓ |

**效果**：用户在登录→注册→忘记密码→重置密码的完整认证流程中，视觉氛围保持连贯，消除「跳页感」。

### 4.2 模块身份辨识度

| 维度 | 优化前 | 优化后 |
| --- | --- | --- |
| 眉题颜色 | 三个模块统一中性灰 | 蓝橙紫各对应模块色 |
| TimeBank 快捷入口 | 翡翠绿（与模块无关） | 紫色（对齐模块色） |

**效果**：用户在不同模块间切换时，眉题色与入口配色即时提示当前所处模块，降低认知成本。设计令牌中预定义的模块色终于被实际消费，设计体系闭环。

### 4.3 Admin 交互体验

| 维度 | 优化前 | 优化后 |
| --- | --- | --- |
| 移动端抽屉 | 瞬间出现/消失 | 滑入 + 淡入过渡 |
| 导航激活态 | 仅背景色 | 背景色 + 左侧色条 |

**效果**：移动端管理员打开侧边栏时有明确的入场动效，导航当前位置有更强视觉锚点。
