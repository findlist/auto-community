## 三、关键流程验证

### 3.1 用户注册 → 首次下单全链路

```
✅ 游客打开首页 → 浏览四大模块
✅ 点击"立即体验" → 进入登录页
✅ 切换到注册页 → 输入手机号、密码、隐私政策勾选
✅ 提交注册 → 跳转首页（已登录态）
✅ 点击"技能" → 浏览列表
✅ 点击某条技能详情 → 展示 AI 推荐
✅ 点击"发起交易" → 成功 → Toast 反馈 → 跳转
✅ 进入"我的订单" → 看到新订单
```

### 3.2 应急场景全链路

```
✅ 用户 A 发布应急求助（位置自动获取）
✅ 推送通知给 300 米内用户 B
✅ 用户 B 收到推送 → 点击进入详情
✅ 用户 B 响应 → 用户 A 收到通知
✅ 用户 A 现场确认完成 → 双方互评
✅ 积分结算 → 信誉分更新
```

### 3.3 时间银行代际互助全链路

```
✅ 父母账号发布家庭绑定邀请
✅ 子女账号收到邀请 → 确认绑定
✅ 时间账户合并（家庭池）
✅ 父母代子女下单（如：接送服务）
✅ 服务完成 → 家庭池时间到账
✅ 父母 → 子女时间转账
```

### 3.4 管理员审核全链路

```
✅ 管理员登录 → 进入 `/admin`
✅ 仪表盘展示核心数据图表
✅ 用户管理 → 封禁/解封操作
✅ 内容审核 → 上下架操作
✅ 订单管理 → 退款/强制完成
✅ 举报管理 → 审核虚假举报
```
