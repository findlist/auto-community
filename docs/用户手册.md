# 「邻里圈」用户手册

> 面向最终用户的操作指南，覆盖从注册到日常使用的全流程。

---

## 一、产品简介

「邻里圈」是一个 AI 驱动的社区互助平台，提供 **技能交换、共享厨房、时间银行、应急邻里** 四大核心服务，让邻里之间互帮互助、资源共享。

### 核心特色
- 🤖 **AI 智能匹配**：基于语义、距离、信誉三维推荐
- 🏘️ **真实场景**：解决生活中真实的邻里互助需求
- 👨‍👩‍👧 **代际互助**：家庭时间池，全家共享
- 🆘 **应急响应**：300 米智能推送，关键时刻靠邻居

## 二、注册与登录

### 2.1 注册账号
1. 打开 App 或网站首页
2. 点击右上角"**注册**"
3. 输入手机号（11 位）
4. 设置密码（≥6 位）
5. 勾选隐私政策
6. 点击"**注册**"完成

### 2.2 登录
1. 点击"**登录**"
2. 输入手机号与密码
3. 登录成功自动跳转到首页

### 2.3 忘记密码
1. 登录页点击"**忘记密码？**"
2. 输入注册手机号
3. 接收并输入验证码
4. 设置新密码
5. 重新登录

### 2.4 个人资料
- 头像：点击"**我的**" → 头像 → 上传新头像
- 昵称：进入个人资料页修改
- 实名认证："**我的**" → "**实名认证**"（增加信誉分）

## 三、技能交换

### 3.1 我能做什么
- **发布技能**：教钢琴、修电脑、辅导功课等
- **寻找技能**：找邻居帮忙

### 3.2 发布技能帖子
1. 进入"**技能**" → 点击"**+**"
2. 填写：
   - 类型：提供 / 需求
   - 分类：教育 / 维修 / 家政 / 搬运 / 医疗 / 宠物
   - 标题（≤30 字）
   - 详细描述
   - 积分定价
3. 上传图片（可选，最多 9 张）
4. 点击"**发布**"

### 3.3 浏览与下单
- 在"**技能**"列表浏览
- 点击进入详情
- 可查看"**AI 智能推荐**"（可能感兴趣的人）
- 点击"**发起交易**"下单

### 3.4 完成交易
- **买家**：联系卖家 → 接受服务 → 确认完成 → 评价
- **卖家**：接受订单 → 提供服务 → 等待买家确认 → 收到积分

### 3.5 积分规则
- 完成订单后积分从买家转到卖家
- 用户最低保留 10 积分
- 注册奖励 100 积分

## 四、共享厨房

### 4.1 我能做什么
- **分享美食**：自制的菜肴、烘焙
- **拼单**：团购便宜实惠
- **发现美食**：地图找附近美食

### 4.2 发布美食
1. 进入"**厨房**" → 点击"**+**"
2. 选择类型：分享 / 拼单
3. 填写名称、描述、价格、人数上限
4. 上传美食图片
5. 选择地址（必填）
6. **健康提示**：请确保已取得健康证

### 4.3 参与拼单
- 查看"**拼单**"列表
- 点击感兴趣的拼单
- 点击"**参与拼单**"
- 拼单成功 → 等待美食

### 4.4 注意事项
- 有过敏原的请务必在"**过敏原**"字段标注
- 拼单失败自动退款

## 五、时间银行

### 5.1 我能做什么
- **存时间**：提供服务赚取时间积分
- **取时间**：消耗时间积分获取服务
- **家庭时间池**：全家共享

### 5.2 时间积分说明
- 时间单位：分钟
- 1 小时 = 60 分钟时间积分
- 注册奖励：60 分钟
- 日收益上限：480 分钟（8 小时）

### 5.3 发布服务
1. 进入"**时间银行**" → 点击"**+**"
2. 选择类型：提供服务 / 需求服务
3. 填写：
   - 服务名称
   - 类别：陪诊 / 看护 / 接送 / 教学 / 维修
   - 时长（分钟）
   - 详细描述
4. 点击"**发布**"

### 5.4 接受订单
- "**我的订单**" → 待响应 → 接受
- 提供服务 → 等待对方确认
- 服务完成 → 收到时间积分

### 5.5 家庭绑定（代际互助）
1. 进入"**时间银行**" → "**家庭绑定**"
2. 输入家人手机号
3. 等待家人确认
4. 绑定成功后：
   - 时间余额共享
   - 可代对方发布需求
   - 适合：父母帮孩子、孩子帮老人

### 5.6 时间转账
- 在"**我的账户**"点击"**转账**"
- 输入接收方手机号、分钟数
- 确认转账

## 六、应急邻里

### 6.1 发起应急求助
1. 进入"**应急**" → 点击"**+**"
2. 填写：
   - 紧急程度：紧急 / 重要 / 一般
   - 求助类型：医疗 / 维修 / 搬运 / 其他
   - 求助内容
   - 联系电话（仅响应者可见）
3. 选择位置（必填）
4. 点击"**发布**"
5. 系统会**自动推送给 300 米内**的响应者

### 6.2 响应求助
- 收到推送通知
- 查看详情
- 点击"**响应**"
- 现场提供帮助
- 等待求助者确认完成
- 完成后获得积分奖励

### 6.3 虚假举报
- 点击"**举报虚假**"
- 管理员审核，核实后扣减对方信誉分

### 6.4 应急资源地图
- 在应急页面查看附近资源
- 包含：社区医院、警务室、消防站等
- 一键导航

## 七、个人中心

### 7.1 我的资料
- 头像、昵称、手机号（已脱敏）
- 实名认证状态
- 加入时间

### 7.2 信誉分
- 范围：0-5 分
- 影响因素：
  - 完成的互助数量
  - 收到的评价
  - 响应时间
  - 违约记录
- 用途：高信誉分用户在 AI 推荐中权重更高

### 7.3 积分流水
- "**我的**" → "**积分流水**"
- 查看所有积分变化记录
- 包含：充值、消费、转账、奖励

### 7.4 通知中心
- 顶部 🔔 入口
- 包含：订单状态、求助响应、举报结果
- 已读/未读管理

### 7.5 设置
- 修改密码
- 隐私设置
- 退出登录
- 账号注销（90 天后彻底删除）

## 八、常见问题

### Q1: 忘记密码怎么办？
A: 登录页点击"忘记密码"，通过手机验证码重置。

### Q2: 交易完成后多久到账？
A: 双方确认完成后即时到账。如超时未确认，系统按规则自动完成。

### Q3: 如何提高信誉分？
A: 按时完成订单、获得好评、积极响应求助都能提升信誉分。

### Q4: 积分可以提现吗？
A: 积分是平台内的虚拟货币，仅用于社区互助消费，暂不支持提现。

### Q5: 注销账号后数据如何处理？
A: 提交注销申请 90 天后，所有 PII 数据将匿名化处理。

### Q6: 应急求助 300 米内没有响应者怎么办？
A: 求助会持续推送 30 分钟，超时后状态变更为"已过期"。

### Q7: 时间积分可以跨家庭使用吗？
A: 家庭时间池内成员共享余额，跨家庭需通过"转账"功能。
